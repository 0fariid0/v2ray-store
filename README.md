# V2Ray Store

Unified install/update command:

```bash
bash <(curl -s https://raw.githubusercontent.com/0fariid0/v2ray-store/main/v2raystore.sh)
```

Default paths:

- Bot: `/var/www/html/v2ray-store`
- Panel: `/var/www/html/v2ray-store-panel`
- Config: `/root/confv2raystore/dbrootv2raystore.txt`

The same command is used for fresh install, update, legacy-folder migration, panel update, webhook repair and panel password change.
